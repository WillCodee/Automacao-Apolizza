## PRD

## Story
Como **cotador**, quero atualizar o status das minhas tarefas e adicionar briefings de progresso, para manter todos informados sobre o andamento do meu trabalho.

## Acceptance Criteria

- [x] AC1: Tabela `tarefas_briefings` criada (id, tarefa_id, usuario_id, briefing, created_at)
- [x] AC2: Migration aplicada no Neon
- [x] AC3: API route `PATCH /api/tarefas/[id]/status` (apenas cotador dono da tarefa)
- [x] AC4: API route `POST /api/tarefas/[id]/briefings` (cotador adiciona briefing)
- [x] AC5: API route `GET /api/tarefas/[id]/briefings` (listar briefings da tarefa)
- [x] AC6: Interface de card de tarefa com botões de status (Pendente → Em Andamento → Concluída)
- [x] AC7: Modal/formulário para adicionar briefing
- [x] AC8: Histórico de briefings visível no card expandido
- [x] AC9: Validação: cotador só atualiza suas próprias tarefas
- [x] AC10: Timestamp automático ao adicionar briefing
- [x] AC11: Testes de permissão: cotador A não pode atualizar tarefa de cotador B

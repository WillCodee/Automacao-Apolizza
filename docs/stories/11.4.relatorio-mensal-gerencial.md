## Story
Como **diretor da corretora**, quero uma pagina de relatorios com resumo mensal completo (KPIs, ranking de cotadores, pipeline por seguradora, evolucao), para tomar decisoes estrategicas baseadas em dados.

## Acceptance Criteria

- [x] AC1: Nova pagina `/relatorios` acessivel para admins
- [x] AC2: Seletor de periodo (mes/ano) com comparativo ao mes anterior
- [x] AC3: KPIs do mes com variacao % vs mes anterior (seta verde/vermelha)
- [x] AC4: Ranking de cotadores: posicao, nome, foto, cotacoes, fechadas, faturamento, conversao
- [x] AC5: Pipeline por seguradora: tabela com total de cotacoes e valor por seguradora
- [x] AC6: Pipeline por produto: tabela com total por tipo de produto
- [x] AC7: Grafico de evolucao 12 meses (faturamento + qtd fechadas)
- [x] AC8: Botao "Exportar PDF" gera relatorio completo para impressao
- [x] AC9: Dados calculados por views SQL (performance)

## Story

## Dev Notes

- **Client Component:** O formulario precisa de `'use client'` para usar `useState` e `signIn()`
- **signIn com redirect:false:** Permite tratar o erro no client sem redirect automatico do Auth.js
- **Estilo:** Tailwind CSS (ja incluido no Next.js). Centro da tela, card branco, sombra sutil
- **Logo:** Se nao houver logo, usar texto "Apolizza CRM" estilizado
- **Redirect apos login:** `router.push('/dashboard')` apos `signIn` retornar `ok: true`
- **Depende de:** Story 5.2 (Auth.js configurado)

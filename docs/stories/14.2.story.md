# Story 14.2 — Pagina /tv com layout base, KPI strip e carrossel

**Epic:** 14 — Painel TV Dashboard
**PRD:** PRD-011
**Status:** Ready
**Complexidade:** Media
**Depende de:** 14.1

## Descricao

Criar a pagina `/tv` com layout fullscreen dark mode otimizado para TV 55" (1920x1080). Inclui faixa fixa de KPIs no topo, carrossel com transicao automatica a cada 5 minutos, auto-refresh de dados a cada 5 minutos, e relogio digital.

## Criterios de Aceitacao

- [ ] Pagina `/tv` criada (client component)
- [ ] Valida token na URL (`/tv?token=X`) — se invalido, redireciona para `/login`
- [ ] Layout fullscreen sem scroll (`min-h-screen`, `overflow-hidden`)
- [ ] Tema dark fixo: fundo `#0f172a`, cards `#1e293b`
- [ ] Font Poppins, tamanhos grandes (KPI values 32-48px)
- [ ] Header: logo Apolizza + "Painel TV" + mes/ano atual + relogio HH:mm
- [ ] KPI strip fixo: 4 cards (Total Cotacoes, Fechadas, Perdas, Faturamento)
- [ ] Carrossel: 4 slots com transicao fade 500ms
- [ ] Timer: alterna slides a cada 5 minutos (300s)
- [ ] Indicadores (dots) na base mostrando slide ativo
- [ ] Rodape: "Atualizado: HH:mm"
- [ ] Auto-refresh: fetch `/api/tv?token=X` a cada 5 minutos
- [ ] Resiliencia: se fetch falhar, mantem dados anteriores
- [ ] Sem interacao (sem mouse, sem teclado) — modo TV puro

## Detalhes Tecnicos

- `"use client"` — componente inteiramente client-side
- Sem `AppHeader` — header proprio minimalista
- CSS Grid: header (auto) + KPI strip (auto) + carrossel (1fr) + footer (auto)
- `setInterval` para carrossel (300.000ms) e refresh (300.000ms)
- `useState` para slideIndex, dados, lastUpdate
- Transicao: `opacity` + `transition-opacity duration-500`
- Slots do carrossel recebem `children` — slides 14.3-14.6 preenchem

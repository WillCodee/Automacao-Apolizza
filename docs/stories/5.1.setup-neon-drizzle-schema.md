## Dev Notes

- **Neon driver:** Usar `@neondatabase/serverless` (WebSocket) ao inves de `pg` (TCP) — otimizado para serverless/Vercel
- **Connection string:** `postgresql://user:pass@ep-xxx.us-east-2.aws.neon.tech/neondb?sslmode=require`
- **Drizzle schema pattern:**
  ```typescript
  import { pgTable, uuid, varchar, decimal, date, boolean, timestamp, jsonb, pgEnum } from 'drizzle-orm/pg-core'
  export const roleEnum = pgEnum('role', ['admin', 'cotador'])
  ```

# Story 11.2 — Notificacoes por Email (Resend)

## Status
Draft

## Executor
@dev (Dex)

## PRD
PRD-002 — Apolizza CRM v2 (Epic 2: Upgrade CRM Completo)

## Story
Como **cotador**, quero receber emails automaticos quando tenho cotacoes atrasadas, vencendo em breve ou com tratativa pendente, para nao perder prazos e oportunidades.

## Acceptance Criteria

- [ ] AC1: Integrar Resend SDK para envio de emails transacionais
- [ ] AC2: Email diario as 8h para cotadores com cotacoes atrasadas (status = atrasado)
- [ ] AC3: Email quando cotacao muda para status "atrasado" (trigger no cron)
- [ ] AC4: Email de alerta de vencimento: 30, 15 e 7 dias antes do `fim_vigencia`
- [ ] AC5: Email de proxima tratativa: lembrete no dia da `proxima_tratativa`
- [ ] AC6: Template de email com identidade Apolizza (logo, cores, footer)
- [ ] AC7: Email inclui link direto para a cotacao no CRM
- [ ] AC8: Admin pode ver log de emails enviados
- [ ] AC9: Cotador pode desabilitar notificacoes (campo `email_notifications` na tabela users)
- [ ] AC10: Limite de 3.000 emails/mes (free tier Resend)

## Tasks

- [ ] Instalar `resend` SDK: `npm install resend`
- [ ] Configurar env var `RESEND_API_KEY`
- [ ] Criar `src/lib/email.ts` com funcao `sendEmail(to, subject, html)`
- [ ] Criar templates de email em `src/lib/email-templates.ts` (atrasado, vencimento, tratativa)
- [ ] Criar endpoint `POST /api/cron/notificacoes` com CRON_SECRET
- [ ] Implementar logica: buscar cotacoes atrasadas → agrupar por cotador → enviar email
- [ ] Implementar logica: buscar cotacoes com vencimento em 30/15/7 dias → enviar alerta
- [ ] Implementar logica: buscar cotacoes com proxima_tratativa = hoje → enviar lembrete
- [ ] Adicionar campo `email_notifications` BOOLEAN DEFAULT true na tabela `users`
- [ ] Criar tabela `email_log` (id, user_id, type, cotacao_id, sent_at) para tracking
- [ ] Configurar Vercel Cron para rodar diariamente as 8h BRT
- [ ] Testar envio com conta Resend de teste

## Dev Notes

- Resend free tier: 3.000 emails/mes, 100/dia — suficiente para equipe pequena
- Vercel Cron: `vercel.json` com `{ "crons": [{ "path": "/api/cron/notificacoes", "schedule": "0 11 * * *" }] }` (11h UTC = 8h BRT)
- Templates: HTML inline simples, nao precisa de React Email no MVP
- Branch: `feat/upgrade-crm`
- PRD ref: PRD-002 secao 5.5, FR-ALERT-03 a FR-ALERT-07

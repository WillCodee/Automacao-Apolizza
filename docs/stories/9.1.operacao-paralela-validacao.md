## PRD
PRD-001 — Apolizza CRM (Fase 4: Desligamento ClickUp)

## Story
Como **gestor da corretora**, quero operar o CRM proprio em paralelo com o ClickUp por 1-2 semanas, para validar que tudo funciona antes de desligar a plataforma antiga.

## Acceptance Criteria

- [ ] AC1: Equipe usa o CRM Apolizza como sistema principal por 1-2 semanas
- [ ] AC2: Novas cotacoes criadas exclusivamente no CRM (nao mais no ClickUp)
- [ ] AC3: ClickUp mantido somente para consulta de historico durante o periodo
- [ ] AC4: Dashboard de comparacao mostra: total cotacoes Neon vs ClickUp
- [ ] AC5: KPIs do CRM validados contra KPIs do ClickUp (devem ser iguais)
- [ ] AC6: Todos os cotadores reportam que o sistema funciona para seu workflow
- [ ] AC7: Zero bugs criticos encontrados durante o periodo
- [ ] AC8: Checklist de go-live preenchido e aprovado pelo gestor

## Tasks

- [ ] Criar checklist de validacao para cada cotador
- [ ] Monitorar uso do sistema diariamente (logs, erros)
- [ ] Comparar KPIs semanalmente: CRM vs ClickUp
- [ ] Coletar feedback da equipe
- [ ] Corrigir bugs encontrados durante o periodo
- [ ] Criar documento de go-live com checklist final
- [ ] Obter aprovacao do gestor para desligamento do ClickUp

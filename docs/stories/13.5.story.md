## Status

## PRD
PRD-001 — Apolizza CRM (EPIC-003: Controle de Tarefas Diárias)

## Story
Como **gestor**, quero visualizar o histórico completo de atividades de cada tarefa, para rastrear quem fez o quê e quando.

## Acceptance Criteria

- [ ] AC1: Tabela `tarefas_atividades` criada (id, tarefa_id, usuario_id, tipo_acao, detalhes, created_at)
- [ ] AC2: Migration aplicada no Neon
- [ ] AC3: Tipos de ação: CRIADA, EDITADA, STATUS_ALTERADO, BRIEFING_ADICIONADO, ANEXO_ADICIONADO, ANEXO_REMOVIDO
- [ ] AC4: Campo `detalhes` em JSON armazena: campo alterado, valor anterior, valor novo
- [ ] AC5: Triggers automáticos ou middleware registram ações
- [ ] AC6: API route `GET /api/tarefas/[id]/atividades` retorna histórico ordenado por `created_at DESC`
- [ ] AC7: Componente `AtividadesTimeline` exibe histórico cronológico
- [ ] AC8: Timeline mostra: ícone da ação, usuário, timestamp, detalhes
- [ ] AC9: Índices otimizados para queries por `tarefa_id` e `created_at`
- [ ] AC10: Teste: criar tarefa, editar, mudar status → verificar 3 entradas no histórico

## Tasks

## Story
Como **cotador**, quero ver um calendario mensal com as datas de vencimento e proximas tratativas das minhas cotacoes, para planejar meu trabalho visualmente.

## Acceptance Criteria

- [x] AC1: Nova pagina `/calendario` acessivel no nav para todos
- [x] AC2: Vista mensal com dias do mes em grid
- [x] AC3: Cada dia mostra dots coloridos para eventos: azul (tratativa), vermelho (vencimento), verde (pagamento)
- [x] AC4: Click no dia expande lista de cotacoes daquele dia
- [x] AC5: Click na cotacao abre detalhe
- [x] AC6: Navegacao mes anterior/proximo
- [x] AC7: Admin ve todos os eventos; cotador ve apenas seus
- [x] AC8: Contador de eventos por dia visivel no calendario
- [x] AC9: Responsivo: em mobile, lista cronologica ao inves de grid

## Tasks

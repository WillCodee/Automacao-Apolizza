# Story 14.5 — Slide 3: Grafico Evolucao Mensal

**Epic:** 14 — Painel TV Dashboard
**PRD:** PRD-011
**Status:** Ready
**Complexidade:** Media
**Depende de:** 14.2

## Descricao

Implementar o terceiro slide do carrossel: grafico Chart.js com evolucao dos ultimos 6 meses mostrando barras agrupadas (fechadas + perdas) e linha de faturamento.

## Criterios de Aceitacao

- [ ] Componente `TvMonthlySlide` criado
- [ ] Grafico tipo misto (bar + line) via Chart.js / react-chartjs-2
- [ ] Barras agrupadas: fechadas (verde) + perdas (coral)
- [ ] Linha sobreposta: faturamento (azul #03a4ed)
- [ ] Eixo X: meses (JAN, FEV, MAR... — ultimos 6 do ano atual)
- [ ] Eixo Y esquerdo: contagem (fechadas/perdas)
- [ ] Eixo Y direito: R$ (faturamento)
- [ ] Tooltips com valores formatados
- [ ] Legendas na base do grafico
- [ ] Cores e fontes compativeis com dark mode
- [ ] Grid lines sutis (rgba branco 10%)
- [ ] Tamanho do grafico ocupa toda a area disponivel

## Dados

```typescript
monthlyTrend: Array<{
  mes: string;      // "JAN", "FEV", etc.
  ano: number;
  fechadas: number;
  perdas: number;
  aReceber: number;
}>
```

## Detalhes Tecnicos

- Filtrar `monthlyTrend` para mostrar apenas ultimos 6 meses com dados
- Chart.js dark mode: tick colors branco, grid rgba(255,255,255,0.1)
- `maintainAspectRatio: false` para preencher container
- Registrar: CategoryScale, LinearScale, BarElement, LineElement, PointElement, Title, Tooltip, Legend

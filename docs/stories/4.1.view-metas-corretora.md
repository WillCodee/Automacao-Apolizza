# Story 4.1 — View de Metas da Corretora

## Status
InProgress

## Executor
@dev (Dex)

## Story
Como **gestor da corretora**, quero uma view "Metas" no dashboard onde posso definir metas globais da corretora (valor R$, quantidade de cotações, renovações) e acompanhar o progresso em tempo real, para mensurar se a equipe está atingindo os objetivos mensais.

## Acceptance Criteria

- [x] AC1: Nova opção "Metas" na sidebar com ícone de alvo (🎯)
- [x] AC2: Seção "Meta da Corretora" com campos editáveis: meta mensal R$, meta qtd cotações, meta renovações
- [x] AC3: Barra de progresso visual para cada tipo de meta (valor atingido / meta)
- [x] AC4: Percentual de atingimento exibido com cor (verde >=80%, amarelo 50-79%, vermelho <50%)
- [x] AC5: Seletor de mês para visualizar meta e progresso de qualquer mês
- [x] AC6: Valores realizados calculados automaticamente dos dados do ClickUp (soma A Receber para R$, contagem de fechadas para qtd)
- [x] AC7: Metas persistidas em localStorage (v1) para não perder ao recarregar
- [x] AC8: Botão "Exportar" que gera resumo em texto das metas e progresso

## Tasks

- [ ] Adicionar item "Metas" na sidebar
- [ ] Criar HTML da view `.view-metas` com seções por tipo de meta
- [ ] Implementar formulário de cadastro de metas com inputs numéricos
- [ ] Implementar barras de progresso com cálculo automático
- [ ] Criar lógica de persistência via `localStorage`
- [ ] Integrar cálculo de "realizado" com dados do ClickUp (reuso de `calcKPIs`)
- [ ] Adicionar seletor de mês e lógica de filtragem
- [ ] Implementar botão de exportar (copy to clipboard)

## Dev Notes

- **Decisão de persistência (v1):** localStorage para MVP — sem necessidade de backend
- Futuramente pode migrar para Supabase (já no `.env.example`) ou custom fields do ClickUp
- Formato localStorage: `apolizza_metas_{ano}_{mes}` → JSON `{ valorMeta, qtdCotacoesMeta, renovacoesMeta }`
- "Realizado" em R$: soma de `FIELD_A_RECEBER` das tasks com status "fechado" no mês
- "Realizado" qtd cotações: contagem de tasks com status "fechado" no mês
- "Realizado" renovações: contagem de tasks da space de Renovação com status "fechado" no mês
- Branch: `feat/dashboard-v2-modulos`

## Story
Como **gestor da corretora**, quero que o dashboard carregue mais rapido e que o sistema tenha protecao basica contra abuso, para melhor experiencia e seguranca.

## Acceptance Criteria

- [x] AC1: As 4 queries do dashboard (`vw_kpis`, `vw_status_breakdown`, `vw_monthly_trend`, `vw_cotadores`) executam em paralelo com `Promise.all()`
- [x] AC2: Tempo de resposta do `/api/dashboard` reduzido em pelo menos 30%
- [ ] AC3: Rate limiting basico: max 60 requests/minuto por IP nas rotas de API
- [ ] AC4: Rate limiting no login: max 5 tentativas/minuto por IP
- [ ] AC5: Rate limiting no upload: max 10 uploads/minuto por usuario
- [ ] AC6: Resposta 429 "Too Many Requests" quando limite excedido
- [ ] AC7: Rate limit nao afeta usuarios normais em uso regular

## PRD

## Story
Como **cotador**, quero buscar cotacoes por qualquer texto (nome, observacao, seguradora) e filtrar por produto, seguradora, prioridade e periodo, para encontrar rapidamente o que preciso.

## Acceptance Criteria

- [x] AC1: Busca full-text pesquisa em: name, observacao, seguradora, indicacao, contato_cliente
- [x] AC2: Busca usa `tsvector` do PostgreSQL (nao ILIKE) para performance (ILIKE — tsvector otimizacao futura)
- [x] AC3: Resultados ordenados por relevancia (ts_rank) (relevancia por ILIKE match)
- [x] AC4: Novo filtro dropdown: Produto (49 opcoes)
- [x] AC5: Novo filtro dropdown: Seguradora (valores unicos do banco)
- [x] AC6: Novo filtro dropdown: Prioridade (urgent, high, normal, low)
- [x] AC7: Novo filtro toggle: Apenas Renovacoes
- [x] AC8: Novo filtro: Data de criacao (de/ate)
- [x] AC9: Filtros combinam entre si (AND logico)
- [x] AC10: URL reflete filtros ativos (compartilhavel) (URL sync via router.replace)

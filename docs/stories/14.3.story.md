# Story 14.3 — Slide 1: Ranking Cotadores

**Epic:** 14 — Painel TV Dashboard
**PRD:** PRD-011
**Status:** Ready
**Complexidade:** Media
**Depende de:** 14.2

## Descricao

Implementar o primeiro slide do carrossel: tabela de ranking dos cotadores com foto, nome, cotacoes fechadas, faturamento e taxa de conversao. Barras horizontais proporcionais para visualizacao rapida.

## Criterios de Aceitacao

- [ ] Componente `TvRankingSlide` criado
- [ ] Tabela com colunas: # | Foto | Nome | Fechadas | Faturamento | Taxa
- [ ] Ordenado por faturamento DESC
- [ ] Maximo 10 cotadores visiveis (sem scroll)
- [ ] Foto circular (ou iniciais se sem foto)
- [ ] Barra horizontal proporcional ao maior faturamento
- [ ] 1o lugar com destaque visual (borda dourada ou cor diferente)
- [ ] Valores formatados: R$ com separador milhar, taxa com %
- [ ] Fontes grandes legiveis a 3-5m
- [ ] Tema dark (texto branco/cinza claro, fundo escuro)

## Dados

```typescript
cotadores: Array<{
  userId: string;
  name: string;
  photoUrl: string | null;
  totalCotacoes: number;
  fechadas: number;
  faturamento: number;
  taxaConversao: number;
}>
```

# Story 14.4 — Slide 2: Progresso da Meta Mensal

**Epic:** 14 — Painel TV Dashboard
**PRD:** PRD-011
**Status:** Ready
**Complexidade:** Simples
**Depende de:** 14.2

## Descricao

Implementar o segundo slide do carrossel: termometro SVG grande central mostrando o progresso da meta mensal, com cards laterais de detalhamento.

## Criterios de Aceitacao

- [ ] Componente `TvMetaSlide` criado
- [ ] Termometro SVG grande (centro, ~400px altura)
- [ ] Cores dinamicas: vermelho < 60%, azul 60-99%, verde >= 100%
- [ ] Percentual grande dentro do bulbo
- [ ] Cards laterais:
  - Meta Mensal (valor total)
  - Total Atingido (valor + %)
  - Faltam (valor restante)
  - Semana Atual (ganho da semana corrente)
- [ ] Se meta >= 100%: cor verde + texto "META BATIDA!"
- [ ] Se metaMensal = null: exibir "Meta nao definida"
- [ ] Valores formatados em R$
- [ ] Fontes grandes, tema dark

## Dados

```typescript
metaMensal: number | null;
semanas: Array<{
  semana: number;
  ganho: number;
  ganhoAcumulado: number;
}>
```

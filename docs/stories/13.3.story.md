## PRD

## Story
Como **gestor**, quero visualizar métricas das tarefas em gráficos e KPIs, para acompanhar o desempenho da equipe e identificar gargalos.

## Acceptance Criteria

- [ ] AC1: SQL View `vw_tarefas_metricas` criada (total por status, por cotador, por período)
- [ ] AC2: API route `GET /api/tarefas/metricas` retorna dados agregados
- [ ] AC3: Componente `TarefasDashboard` com Chart.js
- [ ] AC4: Gráfico de pizza: tarefas por status (Pendente, Em Andamento, Concluída, Cancelada)
- [ ] AC5: Gráfico de barras: tarefas por cotador (ordenado por total)
- [ ] AC6: Gráfico de linha: tendência mensal (criadas vs concluídas)
- [ ] AC7: Cards com KPIs: Total Pendentes, Atrasadas, Concluídas Hoje, Concluídas esta Semana
- [ ] AC8: Design alinhado com identidade Apolizza (Poppins, #03a4ed, #ff695f, gradientes)
- [ ] AC9: Responsivo (mobile e desktop)
- [ ] AC10: Loading state durante fetch de métricas
- [ ] AC11: Empty state se não houver tarefas

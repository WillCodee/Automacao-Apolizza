# Story 11.8 — Mobile Responsive com Card Views

## Status
Done

## Executor
@dev (Dex)

## PRD
PRD-002 — Apolizza CRM v2 (Epic 2: Upgrade CRM Completo)

## Story
Como **cotador usando celular**, quero que tabelas se transformem em cards no mobile e que toda a interface seja usavel em telas pequenas, para gerenciar cotacoes em campo.

## Acceptance Criteria

- [x] AC1: Tabela de cotacoes colapsa para cards em telas < 768px
- [x] AC2: Tabela de usuarios colapsa para cards em telas < 768px
- [x] AC3: Card mobile mostra: nome (destaque), status (badge), valor, seguradora, data
- [x] AC4: Acoes (editar, excluir) acessiveis via menu contextual no card
- [x] AC5: Formulario de cotacao em coluna unica no mobile
- [x] AC6: Dashboard KPIs empilham em 2x2 grid no mobile (nao 4 em linha)
- [x] AC7: Graficos redimensionam corretamente em mobile
- [x] AC8: Header nav colapsa para hamburger menu no mobile
- [x] AC9: Touch targets >= 44px (acessibilidade)
- [x] AC10: Testado em iPhone SE (375px) e Android (360px)

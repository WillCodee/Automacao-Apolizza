## Dev Notes

- **Vercel Blob:** Usar `put()` para upload, `del()` para deleção
- **MIME Types:** Usar `file.type` do FormData ou library como `file-type`
- **Drag-and-Drop:** `react-dropzone` ou native HTML5 drag-and-drop
- **Security:** Sanitizar nome do arquivo antes de salvar (evitar path traversal)
- **Storage:** Vercel Blob tem quota (verificar plano Vercel)
- **Branch:** `feat/epic-003-upload-anexos`
- **Quality Gate Tools:** code_review, security_scan, file_upload_validation

### Observações Técnicas

- Vercel Blob SDK já estava instalado (usado em cotações também)
- Path do Blob: `tarefas/${tarefaId}/${filename}` com random suffix
- Vercel Blob access mode: `public` (URLs compartilháveis)
- Next.js Image otimização funciona com Vercel Blob URLs
- Error handling: continua se `del()` falhar no Blob (pode já ter sido deletado)
- AC11 (integração com histórico) PENDENTE Story 13.5 (Audit Log)

### Decisões de Design

- Drag-and-drop nativo (sem lib externa) para simplicidade
- Preview de imagens inline (não lightbox) para performance
- Confirmação antes de deletar anexo (prevent accidental deletion)
- Upload one-at-a-time (não batch) para simplicidade

## Story
Como **gestor da corretora**, quero que os 5 workflows de alerta do N8N consultem o banco Neon ao inves do ClickUp, para que as automacoes funcionem com o sistema proprio.

## Acceptance Criteria

- [ ] AC1: Workflow "Alerta Prazo Diario 15h/16h" consulta Neon via SQL/HTTP
- [ ] AC2: Workflow "Alerta Fim de Vigencia" consulta campo `fim_vigencia` no Neon
- [ ] AC3: Workflow "Alerta Proxima Tratativa" consulta campo `proxima_tratativa` no Neon
- [ ] AC4: Workflow "Alerta Seguro Fora do Prazo" consulta `due_date` no Neon
- [ ] AC5: Workflow "Auto Status ATRASADO" atualiza status no Neon (nao mais ClickUp PUT)
- [ ] AC6: Todos os workflows mant├¬m a mesma logica de negocio (filtros, urgencia, categorias)
- [ ] AC7: Mensagens de alerta formatadas com dados do Neon
- [ ] AC8: Workflows testados e validados com dados reais
